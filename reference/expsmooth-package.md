# Data sets for "Forecasting with exponential smoothing"

Data sets from the book "Forecasting with exponential smoothing: the
state space approach" by Hyndman, Koehler, Ord and Snyder (Springer,
2008).

## References

Hyndman, R.J., Koehler, A.B., Ord, J.K., and Snyder, R.D., (2008)
*Forecasting with exponential smoothing: the state space approach*,
Springer. <https://robjhyndman.com/expsmooth/>.
